###简要流程：
#####git init（初始化本地仓库）
#####git status（查看当前状态）
#####git add .（添加所有修改的文件到缓冲区）
#####git commit -m ""（提价缓冲区的文件）
#####git remote add origin https://github.com/weiyikang/offer.git （关联远程仓库origin）
#####git push -u origin master（将本地库master提交至远程仓库origin）
